# ADF&G Commercial Fisheries Dataset

## Overview

This dataset contains commercial fish count data from Alaska Department of Fish and Game (ADF&G) monitoring stations across Alaska. The data spans multiple years and includes various salmon species counts from commercial fishing operations.

## Geographic Areas

- **Active Commercial Stations:** 19 monitoring stations across Alaska
- **Species Coverage:** Chinook, Sockeye, Coho, Pink, and Chum salmon
- **Temporal Coverage:** Historical data from commercial fishing operations

## Data Structure

Each location directory contains species subdirectories with year-specific JSON files:
- `[location-id]/[species-id]/[year-location-species].json`

## Species Codes

- **410**: Chinook Salmon
- **420**: Sockeye Salmon  
- **430**: Coho Salmon
- **440**: Pink Salmon
- **450**: Chum Salmon

## Data Source

- **Provider**: Alaska Department of Fish and Game (ADF&G)
- **Dataset**: Commercial Fisheries Monitoring
- **Last Updated**: afca-master-build
- **Total Files**: 41

## Usage

This dataset is designed for use with the Alaska Fish Count App (AFCA) and can be accessed via:
- GitHub Repository: `https://github.com/alaskafishcounts/adfg-commercial-dataset`
- Raw Data Access: `https://raw.githubusercontent.com/alaskafishcounts/adfg-commercial-dataset/master/`
